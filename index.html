How to Host on GitHub Pages:

  1. Create a new GitHub repository named "gps-tracker"
  2. Upload this HTML file to the repository (name it index.html)
  3. Go to repository Settings > Pages
  4. Set source to "Deploy from a branch"
  5. Select branch: main or master
  6. Select folder: /(root)
  7. Click Save - your site will be at:
    https://yourusername.github.io/gps-tracker/

Note: Make sure your Google Sheet is set up to receive data:

  1. Open your Google Sheet: 1TFKdPbSlNWIqSnlja1PRSXaiBZ-vIG85j6XtwRntEbM
  2. Create columns: Timestamp, Latitude, Longitude, Accuracy, Device, User ID
  3. Deploy the script as a web app with permissions set to "Anyone, even anonymous"
